First day on the way to be a coder

Step 1: Installation

Go to website Visual studio code to install VSCode. Next, you may need to link your github account to VSCode, there are several options for installing Git on MacOS.

These are a few ones to choose: Homebrew or Macports to install.

After installing:

  • To Home Brewprint "brew install git"
  • To Macports, print "sudo port install git"

You can find and download it via Git-scm.com
